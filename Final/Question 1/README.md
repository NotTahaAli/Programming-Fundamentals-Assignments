# Question 1
## Marks: 10
## Question Statement
Find the Largest Subarray of 1s in an array of 1s and 0s, such that you ignore exactly one element.
## Examples:
### Example 1
#### Input: `[0,1,1,0,1]`
#### Output: `3`
### Example 2
####  Input: `[1,1,1]`
#### Output: `2`
#### Explanation
As we need to ignore exactly one element, we ignore the first or last element.
